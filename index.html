      Persyaratan Server

      Kerangka kerja Laravel memiliki beberapa persyaratan sistem. Semua persyaratan ini dipenuhi oleh mesin virtual Laravel Homestead, jadi sangat disarankan agar Anda menggunakan Homestead sebagai lingkungan pengembangan Laravel lokal Anda.

      Namun, jika Anda tidak menggunakan Homestead, Anda harus memastikan server Anda memenuhi persyaratan berikut:

      • PHP version 7.2 and later
      • BCMath PHP Extension
      • Ctype PHP Extension
      • JSON PHP Extension
      • Mbstring PHP Extension
      • OpenSSL PHP Extension
      • PDO PHP Extension
      • Tokenizer PHP Extension
      • XML PHP Extension

        Di sini kami memformat tanggal dengan 'j M Y H:i:s A e'

        • j – Hari dalam sebulan tanpa nol di depan( 1 to 31 )
        • M – Representasi tekstual singkat dari satu bulan, tiga huruf ( Jan hingga Dec )
        • Y – Representasi numerik penuh dari satu tahun, 4 digit ( Contoh: 1999 or 2003 )
        • H:i:s – Jam, menit dan detik
        • A – AM / PM
        • e – zona waktu ( default UTC )

        Payment gateways

        Booking Core memungkinkan Anda menggunakan 3 cara pembayaran: Pembayaran offline, Paypal, dan Pay Stripe

        1. OFFLINE PAYMENT

        Untuk mengaktifkan opsi ini, Buka Dashboard > Settings > Payments Settings > Payment Gateway Aktifkan Offline Payment

        • Metode ini memungkinkan pengguna Anda dapat memesan layanan, tetapi mereka tidak perlu membayar segera.
        • Mereka dapat memilih layanan lalu pergi ke Kantor Anda untuk membayar.
        • Untuk mengaktifkan metode ini, Anda hanya perlu Aktifkan Kirim Formulir.

        Anda juga dapat mengubah logo formulir pengiriman.

         

        2. PAYPAL PAYMENT

        • Metode ini memungkinkan pengguna Anda dapat memesan layanan, dan mereka dapat membayar langsung melalui akun Paypal.
        • Untuk mengaktifkan metode ini, Anda hanya perlu Mengaktifkan Paypal
        • Lanjutkan memasukkan informasi paypal Anda.

        Jika Anda memasukkan informasi yang salah, Pemesanan klien tidak dapat dilakukan lagi.

         

        3. PAYSTRIPE PAYMENT

        • Metode ini memungkinkan pengguna Anda dapat memesan layanan, dan mereka dapat membayar langsung melalui akun Stripe 
        • Untuk mengaktifkan metode ini, Anda hanya perlu Mengaktifkan Stripe
        • Lanjutkan memasukkan beberapa informasi sebagai: Kunci Rahasia Stripe Anda,  Aktifkan Mode Sandbox, Your Stripe Secret Key, Enable Sandbox Mode, Your Stripe Test Secret Key for Sandbox mode

        Jika Anda memasukkan informasi yang salah, Pemesanan klien tidak dapat dilakukan lagi.

        Pricing for Tour

        There are 2 ways to calculate the price for Tour: Fixed Price and Price by Type of Person

        1. Fixed Price: Semua dalam harga Tur untuk maksimum / minimum orang yang ditentukan

        - Secara default, Harga Tur akan dihitung sesuai dengan harga yang Anda masukkan di bagian Harga Tur.

        - Setiap pemesanan tour akan menjadi Inclusive tour dengan jumlah orang dalam rentang min / max yang disebutkan di bagian konten Tour, itu berarti jumlah orang tergantung pada Anda, selama dalam rentang yang diizinkan

        - Bagian yang menunjukkan persentase diskon akan selalu ditampilkan pada harga di harga Tur

         

        2. Price by Type of Person: Centang kotak untuk mengaktifkan harga per orang

        Bila Anda mengaktifkan opsi ini, semua harga hanya akan dihitung sesuai dengan harga yang Anda tawarkan untuk setiap objek:

        • Anak-anak: peraturan dari 6-15 tahun - harga $560 , hanya mengizinkan minimal 3 orang dan maksimal 10 orang
        • Dewasa: peraturan dari 16-50 tahun - harga $560...
        • Orang tua: peraturan dari 50-60 tahun - harga $560...

         

        Extra service(Opsional): Anda dapat menambahkan lebih banyak layanan untuk Tur Anda seperti: Makanan Cepat Saji, Minuman..

        Anda dapat mengikuti langkah-langkah di bawah ini:

        • Langkah 1: Centang kotak untuk Mengaktifkan Harga Tambahan
        • * Namun Anda perlu INGAT: Jika Anda Mengaktifkannya tetapi Anda tidak memasukkan apa pun, pemesanan Tur Anda akan GAGAL
        • Langkah 2: Masukkan judul Layanan ekstra
        • Langkah 3: Masukkan harga layanan
        • Langkah 4: Pilih Jenis layanan: harga per Satu kali atau harga per Orang

         

        Discount type :

        • Cart discount – Diskon total tetap untuk seluruh keranjang. Misalnya, jika kereta berisi tur, kupon diskon $10 akan memberikan diskon $10.
        • Cart % discount –  persentase diskon untuk seluruh troli. Misalnya, jika troli berisi tur @ $60, kupon diskon 10% akan memberikan diskon $6.

    Multi-languages ¶

    Dari versi 1.2, sistem Booking Core kami mendukung untuk fitur multi-bahasa, sehingga Anda dapat dengan mudah menggunakan beberapa bahasa di situs web Anda. Berikut beberapa pedomannya:

    1. Enable Multi-Language

    Pertama, arahkan Dashboard -> Settings -> General Settings -> Language :

    1. Select Main language: Ini adalah langkah penting untuk membangun konten situs web Anda. Karena sub-bahasa hanya dikompilasi ulang dari bahasa utama, jika Anda menggunakan EN sebagai bahasa utama dan tiba-tiba beralih ke JA, situs web Anda akan gagal. Karena tidak ada konten utama JA yang digunakan.

    2. Langkah selanjutnya, Enable Multi languages: Centang kotak Enable Multi-languages untuk situs

    3. Setelah mengaktifkan Multi-languages, Anda akan melihat situs web Anda terdapat tampilan Language bar

     

    2. Start translate for site
